# 🛡️ Desafio DIO - Classificador de Nível de Herói

Projeto desenvolvido durante os estudos de lógica de programação na **DIO** (Digital Innovation One).

## 📋 Sobre o desafio

O objetivo é criar um programa que recebe o nome e a quantidade de experiência (XP) de um herói e retorna qual o nível correspondente, de acordo com as faixas abaixo:

| XP | Nível |
|---|---|
| Menor que 1.000 | Ferro |
| 1.001 a 2.000 | Bronze |
| 2.001 a 5.000 | Prata |
| 5.001 a 7.000 | Ouro |
| 7.001 a 8.000 | Platina |
| 8.001 a 9.000 | Ascendente |
| 9.001 a 10.000 | Imortal |
| Maior ou igual a 10.001 | Radiante |

## 💻 Conceitos utilizados

- Variáveis
- Operadores
- Laços de repetição
- Estruturas de decisão (if / else if)
